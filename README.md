# PS3-GENJI
PS3　GENJIの分解
Internal character nicknames found in PS3 GENJI (BPAK format)
​I discovered that character data in PS3 GENJI is categorized by internal nicknames within the binary.
By searching for the keyword "Model" inside .BPAK files, you can identify these nicknames (e.g., "SHIZU" for Shizuka-gozen or "YOSSY" for Yoshitsune) and successfully locate the specific character data offsets.
​It's fascinating to see these "dev-team only" nicknames still alive in the code after all these years!
PS3 GENJIの解析において現時点で分かっていることはBPAK形式であり、キャラデータは愛称で呼ばれていることです。静御前ならShizu、義経ならYOSSY、敵キャラはEから始まるコード、味方キャラはC0の番号で呼ばれています
